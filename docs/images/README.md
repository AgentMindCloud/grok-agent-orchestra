# docs/images

Drop GIFs and screenshots here.

Suggested files referenced from the top-level `README.md`:

- `tui-demo.gif` — Rich live TUI showing a debate stream, mid-loop veto, and final verdict.
- `architecture.png` — Optional static rendering of the Mermaid architecture diagram.
- `cli-init.gif` — `grok-orchestra init` + `run --dry-run` happy path.

Keep individual assets under ~3 MB so they load on the GitHub README without
the user staring at a spinner. Prefer GIF/PNG over MP4 — GitHub renders the
former inline.
